    写作背景

    在Android源码实现部分，很多人都应该分析过View的绘制流程，measure，layout，draw三个过程也许已经十分熟悉了，但是相信有很多小伙伴和笔者一样并不知道到xml布局到底是如何被解析然后转换成View的，今天笔者将和大家一起来学习这个流程(基于Android API 28源码)。
